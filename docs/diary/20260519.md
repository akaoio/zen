## 1. Bug fixes trong `adopt()` (script/server.js)

Hai bug được xác định và fix:

### Bug 1: PEX broadcast dùng sai scheme

```js
// Cũ — sai
route.say({ dam: "pex", peers: [n] }, ...);
// n = "https://..." — receivers check /^wss?:\/\// → silently ignored!

// Mới — đúng
route.say({ dam: "pex", peers: [PeerRegistry.alt(n)] }, ...);
// PeerRegistry.alt() = "wss://..." — receivers accept ✅
```

### Bug 2: Reconnecting peer không được re-gossip

```js
// Cũ
if (registry.has(n)) return; // ← khi relay reconnect, không gossip lần nào nữa

// Mới
if (registry.has(n)) {
  if (!gossiped.has(n) && route) {
    gossiped.add(n);
    route.say({ dam: "pex", peers: [PeerRegistry.alt(n)] }, ...); // gossip 1 lần/session
  }
  return;
}
```

**759 tests pass** sau hai fix này.

implement sai (scobru, và nhiều người khác).

## 3. Điều tra relay IP-only (không có domain)

`originip` hoàn toàn thiếu trong code cũ:
- `disc()` phát hiện IPv4 nhưng chỉ lưu vào `discResult` (cho `/status`)
- `domain` variable không bao giờ được cập nhật từ `disc()` result
- Relay không có domain = **vô hình với mesh gossip**

`disc()` có `if (!dom && ip) { dom = ip; }` nhưng chỉ trong returned object, không phải server.js's domain variable.

## 4. Refactor chính: `lib/pex.js`

Tạo `lib/pex.js` — hàm `setupRelayPex(zen, opts)` đóng gói toàn bộ relay-side PEX:

### API

```js
const { origin, registry, adopt, getSelfUrl } = setupRelayPex(zen, {
  domain,          // "zen.akao.io" hoặc null (IP-only)
  port,            // 8420
  key,             // TLS key → wss:// vs ws://
  registry,        // PeerRegistry instance
  pexMax: 50,      // max peers in PEX list
  rttOf,           // fn(url) → RTT ms for sorting
  sendPeers,       // fn(list, peer) — override default mesh.say (UDP fast-path, bpids)
  onDisc,          // fn(di) — called after each disc() run
  onAdopt,         // fn(url) — called when NEW peer added to registry
});
```

### Những gì `setupRelayPex` xử lý

| Feature | Mô tả |
|---------|-------|
| `origin` | Domain-based self-URL, stable |
| `originv` | IPv6 self-URL, auto-updated mỗi 10 phút |
| `originip` | **IPv4 self-URL — MỚI**, cho relay không có domain |
| `mesh.hear["pex"]` | Nhận peer list từ các relay khác |
| `root.on("hi")` | Gửi peer list + self-URL khi peer mới connect |
| Re-gossip | Reconnecting peers được gossip 1 lần/session |
| wss:// fix | Broadcast dùng `PeerRegistry.alt()` |

### Hooks để server.js customize

- `sendPeers`: server.js dùng `saySmart` (UDP fast-path) + bpids cho browser WebRTC
- `onDisc`: cập nhật `discResult` cho `/status`
- `onAdopt`: set `found = true` + gọi `probe()` cho scan pattern expansion
